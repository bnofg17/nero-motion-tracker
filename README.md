## 三种追踪模式，各有各的用场

### 1. 分段追踪（Segment Tracking）——2025 年新上的主角

这是最近一次大更新的核心功能，也是和其他追踪软件拉开差距的地方。

传统的追踪是画一个矩形框框住目标，但现实里的物体哪有那么规整？一只跑动的狗、一个跳舞的人，轮廓一直在变，矩形框要么框不住，要么遮太多。

分段追踪直接识别物体轮廓，生成帧帧自适应的动态遮罩，就算目标被遮挡一会儿、换了个角度，也能继续跟上。用来给奔跑的人物打码，或者给特定物品添加跟随特效，效果明显比普通追踪精准。

### 2. 人脸追踪（Face Tracking）

自动识别视频里的人脸，不管目标转头、侧脸、被短暂遮挡，都能持续跟随。一键对检测到的人脸施加马赛克或模糊效果，街拍、采访、Vlog 场景最常用到。

### 3. 区域追踪（Zone Tracking）

你手动圈定一个区域，软件负责跟着它动。速度比分段追踪快，还支持 NVIDIA GPU 加速，适合处理车牌、路牌等形状规则的目标，或者就是单纯想快速遮住某个固定区域。
